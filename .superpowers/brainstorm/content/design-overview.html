Phase 3.12 — Design Overview

4 ส่วนหลัก — อ่านแล้วบอกว่าโอเค หรือปรับตรงไหน

① Permissions เพิ่มใหม่ (Permissions.php)

// employees module เพิ่ม 3 keys
'employees' => [
  'view', 'add', 'edit', 'deactivate', 'edit_own',
  'reset_password',  'resign',  'set_credentials'
]
• reset_password — ควบคุมปุ่ม Reset Password ใน Employee Profile
• resign — ควบคุมปุ่ม Resign ใน Employee Profile
• set_credentials — รับ notification + ตั้ง Username/Password ให้ Employee ได้

② Employee Create → User Account Flow

✅ มี Email
1. สร้าง User อัตโนมัติ
2. password = Employee Code
3. User.role = role ของ GroupRole
4. 🔔 Notify IT: "Credentials ready — default: EMP-XXXX"
5. 📧 Email IT ด้วย
⚠️ ไม่มี Email
1. ยังไม่สร้าง User
2. 🔔 Notify IT: "ต้องตั้ง Username/Password เอง"
3. 📧 Email IT ด้วย
4. IT ไปที่ Employee Profile → ตั้ง Credentials
5. User ถูกสร้างตอนนี้

④ GroupRole → User.role Auto-Sync

• เมื่อ Employee เข้า GroupRole → User.role = group.role
• เมื่อ Employee ออกจาก GroupRole → User.role = 'user' (default)
• ตอน Create Employee + มี Default Group → assign เข้า group + sync role ทันที
• Login รองรับ email หรือ username (สำหรับ Employee ไม่มี Email)
